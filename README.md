# SF_RateDialog

This is a quick and simple library to send a dialog that asks the user to rate the app.

Dependency
build.gradle (root)

	allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
build.gradle (app)
  
  	dependencies {
	        implementation 'com.github.SteinFluss:SF_RateDialog:0.1'
	}
